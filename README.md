Drop Timer Manager

ゲームのドロップ時間を管理するためのタイマーWebサイトです。

ステージ名と時間を入力して、複数のタイマーを管理できます。

公開URL

https://horiguchi-rr.github.io/drop-timer/

主な機能

ステージ名を入力してタイマーを追加
分・秒を指定してカウントダウン
Start / Reset / Delete ボタン
00:00になると「ドロップ可能！」と表示
ページを更新してもタイマーを保存
ページを閉じていても終了予定時刻から残り時間を計算
スマホ表示対応

使用技術

HTML
CSS
JavaScript
localStorage
GitHub Pages

使い方

ステージ名を入力する
分と秒を入力する
New Timer ボタンを押す
Start ボタンでカウント開始
Reset ボタンで初期時間に戻す
Delete ボタンでタイマーを削除する

工夫した点
localStorage を使って、ページ更新後もタイマーを保存できるようにしました。
終了予定時刻を保存することで、ページを閉じている間も時間が進むようにしました。
ステージ名の表示にはエスケープ処理を入れ、HTMLとして実行されないようにしました。
スマホでも使いやすいようにレスポンシブ対応しました。

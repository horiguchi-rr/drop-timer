# Drop Timer Manager

ゲームのドロップ時間を管理するためのタイマーWebサイトです。

ステージ名と時間を入力して、複数のタイマーを管理できます。

## 公開URL

https://horiguchi-rr.github.io/drop-timer/

## 主な機能

- ステージ名を入力してタイマーを追加
- 分・秒を指定してカウントダウン
- Start / Reset / Delete ボタン
- 00:00になると「ドロップ可能！」と表示
- ページを更新してもタイマーを保存
- ページを閉じていても終了時刻をもとに残り時間を計算
- スマホ表示対応

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages

## 使い方

1. ステージ名を入力する
2. 分と秒を入力する
3. New Timer ボタンを押す
4. Start ボタンでカウント開始
5. Reset で初期時間に戻す
6. Delete でタイマーを削除する
